## 数组的理论知识
当给数组分配内存时，数组中的各个元素是相邻的，所以可以通过下标来访问其中的某个元素。

```text
let nums = [10, 11, 12, 13, 14, 15];

Array:   10  11  12  13  14  15
index:    0   1   2   3   4   5

assert_eq(nums[0], 10);
assert_eq(nums[5], 15);
assert_eq(nums[6], xx); // ERROR，index out of bounds
```
